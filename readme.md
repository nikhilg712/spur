# Spur AI Live Chat Agent

# Tech
- Backend: Node.js, TypeScript, Express
- DB: SQLite + Prisma
- Frontend: SvelteKit
- LLM: OpenAI

# Setup
1. Clone repo
2. cd backend
3. npm install
4. add .env with OPENAI_API_KEY
5. npx prisma migrate dev
6. npm run dev

7. cd frontend
8. npm install
9. npm run dev
